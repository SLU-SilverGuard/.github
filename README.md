# SLU-SilverGuard Organization Profile Repo

这个仓库是 GitHub 组织级特殊仓库 `.github`。

## 用途

- `profile/README.md` 用于组织主页展示
- 根 `README.md` 用于说明这个特殊仓库的维护方式

## 当前结构

- `profile/README.md`
  - 组织主页主文案
  - 介绍 4 个核心仓库的关系、目标、技术路线和亮点

## 维护约定

- 当 `backend`、`companion`、`guardian`、`frontend` 的定位、技术栈或协作关系变化时，同步更新这里
- 组织主页文案应优先描述“为什么做、解决什么问题、4 个仓库如何协作”
- 详细开发约束仍分别维护在 4 个业务仓库自己的 `README.md` 和 `AGENTS.md`
